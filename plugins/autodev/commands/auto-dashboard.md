# TUI 대시보드 (/auto-dashboard)

ratatui 기반 터미널 대시보드를 실행합니다.

## 실행

```bash
autodev dashboard
```

## 키바인딩

| 키 | 동작 |
|----|------|
| Tab | 패널 전환 (Repos → Queues → Logs) |
| j/k | 목록 상/하 이동 |
| Enter | 상세 보기 |
| r | 실패 항목 재시도 |
| q | 종료 |
| ? | 도움말 |
