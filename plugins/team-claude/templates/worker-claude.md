## Guidelines

### Before Starting

1. **Check for feedback**: Read `.claude/feedback.md` if it exists
2. **Understand the spec**: Read `.claude/task-spec.md` carefully
3. **Explore context**: Look at related files in the codebase

### While Working

1. **Stay focused**: Only implement what's in the spec
2. **Commit often**: Make small, focused commits
3. **Test as you go**: Don't leave testing for the end
4. **Document blockers**: If stuck, write to `.claude/blockers.md`

### Commit Messages

Use conventional commits format:
```
feat: add user authentication endpoint
fix: resolve null pointer in payment handler
test: add unit tests for auth service
refactor: extract validation logic to helper
```
